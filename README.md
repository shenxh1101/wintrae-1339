# 📝 网课学习笔记助手 (Browser Extension)

一款帮助网课学习者高效整理视频学习重点的浏览器扩展。支持侧边笔记、时间戳书签、截图标注、复习列表和设置面板等功能。

## ✨ 功能特性

### 📝 侧边笔记
- 观看视频时随时记录当前时间点笔记
- 支持自定义标签分类
- 搜索和筛选历史笔记
- 一键跳转回笔记对应视频位置

### 🔖 时间戳书签
- 快速添加当前播放位置书签
- 按课程、章节和标签多层分类
- 点击书签自动跳转回对应视频位置
- 支持编辑书签标题、描述和标签

### 📷 截图标注
- 一键截取当前视频画面
- 内置画布标注工具：
  - ➡️ 箭头标注
  - ⬜ 矩形框选
  - 🖍️ 高亮区域
  - ✏️ 文字标注
- 支持颜色和粗细调整
- 撤销操作支持

### 🔁 复习列表
- 汇总待复习学习片段
- 标记「待复习」和「已掌握」状态
- 自定义复习提醒时间
- 过期提醒高亮显示
- 直接从书签快速添加复习项

### ⚙️ 设置面板
- 显示/隐藏浮动快捷窗口
- 自动检测视频配置
- 复习提醒天数设置
- 快捷键说明
- 数据导出（JSON / Markdown）
- 清理无效课程记录
- 清空所有数据

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+N` | 快速记录当前时间点笔记 |
| `Ctrl+Shift+B` | 快速添加当前时间点书签 |
| `Ctrl+Shift+S` | 快速截取当前画面 |
| `Ctrl+Shift+P` | 打开/关闭侧边栏 |

> 💡 可在 `chrome://extensions/shortcuts` 中自定义快捷键

## 📦 安装方法

### 开发者模式安装（推荐）

1. 打开 Chrome / Edge 浏览器，进入扩展管理页面：
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`

2. 打开右上角的「开发者模式」开关

3. 点击「加载已解压的扩展程序」按钮

4. 选择本项目所在的文件夹（`d:\TraeProjects\1339`）

5. 扩展安装完成，工具栏会出现扩展图标 ✅

## 🚀 使用方法

### 首次使用
1. 打开任意视频网站（如 B站、YouTube、腾讯课堂、网易云课堂等）
2. 页面右侧会出现浮动快捷窗口（可拖动位置）
3. 点击扩展图标打开侧边栏，或按 `Ctrl+Shift+P`

### 记录学习笔记
1. 视频播放到需要记录的位置
2. 按 `Ctrl+Shift+N` 或点击浮动窗口的「笔记」按钮
3. 在弹窗中输入笔记内容和可选标签
4. 点击「保存」即可

### 添加时间戳书签
1. 视频播放到重点位置
2. 按 `Ctrl+Shift+B` 或点击浮动窗口的「书签」按钮
3. 书签自动保存，可在侧边栏中编辑详情

### 截取并标注画面
1. 视频播放到需要截图的画面（可暂停）
2. 按 `Ctrl+Shift+S` 或点击浮动窗口的「截图」按钮
3. 在标注编辑器中使用工具标注重点
4. 填写标题和描述，点击「保存」

### 管理复习计划
1. 在侧边栏切换到「复习」标签页
2. 点击「➕ 添加」按钮创建复习项
3. 设置下次复习日期和状态
4. 也可以在书签列表中点击「🔁」按钮直接将书签加入复习

## 🛠️ 数据存储

所有数据保存在浏览器本地 `chrome.storage.local` 中，包括：
- 课程信息（URL、标题、章节）
- 笔记内容、时间戳、标签
- 书签及其元数据
- 截图原图和标注后的图片
- 复习计划及状态

## 📤 数据导出

在侧边栏「设置」面板中可以：
- **导出全部数据 (JSON)**：完整备份所有数据，用于迁移或恢复
- **导出笔记 (Markdown)**：导出为人类可读的 Markdown 格式

## 🔧 重新生成图标

如果需要重新生成图标，有两种方式：

### 方式一：PowerShell 脚本
```powershell
./generate-icons.ps1
```

### 方式二：浏览器工具
在浏览器中打开 `generate-icons.html`，点击下载按钮分别保存三种尺寸的图标到 `icons/` 目录。

## 📝 注意事项

1. 扩展仅在浏览器本地运行，所有数据不会上传到任何服务器
2. 清理浏览器数据会同时清除扩展数据，建议定期导出备份
3. 部分网站有 CSP 策略，截图功能可能会受影响
4. iframe 中的视频在跨域情况下可能无法自动检测

## ❓ 常见问题

**Q: 浮动窗口没有出现？**
A: 请检查设置中「显示浮动窗口」是否开启，或按 `Ctrl+Shift+P` 打开侧边栏查看设置。

**Q: 点击时间戳无法跳转？**
A: 确保视频页面仍在浏览器中打开，且扩展已获得该页面的访问权限。

**Q: 截图是黑屏？**
A: 部分 DRM 保护的视频禁止截图，这是浏览器的安全限制。可尝试使用系统截图工具后手动导入。
